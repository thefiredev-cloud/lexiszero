# Lexis Zero

**Legal Intelligence, Zero Billable Hours**

AI-powered legal analysis for contracts, compliance, and case research at a fraction of the cost.

## Features

- **Contract Analysis** — Automatically extract key clauses, identify risks, and highlight obligations
- **Case Research** — Search through millions of precedents with AI-powered relevance ranking
- **Compliance Monitoring** — Stay ahead of regulatory changes across jurisdictions
- **Risk Assessment** — Proactive identification of legal risks and potential liabilities

## Tech Stack

- Next.js 14 (App Router, static export)
- React 18
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## Build

```bash
npm run build
```

Generates a static export in the `out/` directory.
